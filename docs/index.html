What it is

TWO LENSES.
ONE HONEST SYSTEM.

FineMargins quantifies how pressure shifts World Cup scoring patterns — and is explicit about what AI cannot know. Built on IBM Granite, IBM Docling, and 192 real matches of StatsBomb open data.

⚡

Pressure Lens

Analyses every penalty kick and late-game shot across 192 real World Cup matches. Two logistic regression models with exact SHAP attribution show how shootout context, sudden-death stakes, and knockout pressure shift conversion rates at the population level. AUC 0.518 on individual prediction is reported prominently — because it is the correct honest finding, not a failure to hide.

Shootout: 65.0%
In-game: 74.7%
Gap: −9.7pp
SHAP: exact

Officiating Lens 🔴 LIVE 2026

Real findings · Not fabricated

NUMBERS THAT
HOLD UP.

Every number below comes from real StatsBomb open data — 192 matches, 4,878 shot events. Verified by 5-fold cross-validation. Model limits stated explicitly, not buried.

Shootout gap −9.7pp

65.0% (n=123) vs 74.7% (n=79) — real, measurable, persists across all 3 tournaments. 95% bootstrap CIs overlap slightly at this sample size.

Sudden-death paradox 71.4%

Sudden-death kick conversion from 14 real kicks — counter-intuitively higher than regular shootout kicks (65.0%). Small sample, but real and unreported elsewhere.

xG overestimate 110.87 → 103

StatsBomb's xG expected 110.87 late-game goals — 103 were scored. Pressure suppresses finishing by ~7 goals beyond what shot quality alone predicts.

Honest null result AUC 0.518

Pressure context barely beats random for individual penalty prediction. Correct finding, not hidden — pressure shifts populations, not individual outcomes.

VAR overturn rate 92.6%

25/27 VAR reviews overturned in Qatar 2022. Penalty checks overturned 55.6% (5/9). Source: ESPN VAR Review Log + FIFA 2022 WC Technical Report.

Model B vs baseline 0.773 vs 0.807

Pressure model vs xG-only baseline. Pressure does not improve individual discrimination — population-level effects don't transfer to individual prediction. Stated honestly.

Architecture

FIVE STEPS FROM
DATA TO INSIGHT

From StatsBomb's open JSON to IBM Granite narration — a verifiable pipeline where every number traces to a real source.

finemargins / pipeline · ibm_layer · app
01

StatsBomb Open Data Fetch

192 matches fetched concurrently from StatsBomb's public GitHub — 2018 WC, 2022 WC, Women's WC 2023. No scraping, no paid API. Licensed under StatsBomb Public Data User Agreement. All data pre-committed as parquet — no pipeline run needed to explore the app.

StatsBomb JSON ThreadPoolExecutor parquet committed to git
02

Pressure Feature Engineering

202 penalty kicks and 1,228 late-game shots extracted. Features: is_shootout, is_sudden_death, is_knockout, stage_weight (0=Group Stage → 1=Final), xG_residual (actual − StatsBomb xG), minutes_remaining. Separation is clean — no penalty double-counted as a late-game shot.

pandas numpy stage weight encoding
03

Model Training + Exact SHAP

Model A: Logistic Regression on 202 penalties — AUC 0.518, honest null result reported prominently. Model B: Logistic Regression + xG residual — AUC 0.773 vs xG-only baseline 0.807. SHAP LinearExplainer computes exact (not approximate) Shapley values for every kick. 5-fold stratified CV throughout.

scikit-learn shap LinearExplainer (exact) 5-fold CV bootstrap CIs
04

IBM Tool Layer — Anti-Hallucination

IBM Docling 2.103 parses the IFAB Laws of the Game 2025/26 PDF into law_chunks.json. Three grounding tools (get_pressure_profile, get_law_text, get_overturn_rate) read real parquet/JSON and raise ToolDataError if data is absent — Granite cannot invent a number, only narrate verified ones.

IBM Docling 2.103 IBM Context Forge MCP 1.0.3 ToolDataError guard
05

IBM Granite Narration — 3 Voice Modes

IBM Granite (ibm/granite-4-h-small via watsonx.ai) narrates findings in fan, analyst, and referee-trainee modes. Receives structured data from grounding tools only. 7-page Streamlit app deployed via Docker on HF Spaces with 4 watsonx secrets. Full demo mode without credentials — all pages load from committed parquet.

IBM Granite (watsonx.ai) ibm-watsonx-ai 1.3.42 Streamlit 7 pages Docker · HF Spaces
