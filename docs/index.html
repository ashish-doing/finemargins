// What it is

TWO LENSES.
ONE TRUTH.

FineMargins is a World Cup pressure intelligence system with two analytical lenses — both grounded in real StatsBomb open data, explained by IBM Granite, and honest about what they cannot know.

⚡

Pressure Lens

Analyses every penalty kick and late-game shot across 192 real World Cup matches to quantify how pressure context shifts conversion probability — and then is honest about what it can and cannot predict.

Shootout: 65.0%
In-game: 74.7%
Gap: −9.7pp
🟨

Officiating Lens

Explains 5 real, documented VAR incidents from World Cup history using actual IFAB Laws parsed by IBM Docling, real historical base rates, and an explicit "what this system cannot know" panel per incident.

VAR reviews: 27
Overturned: 92.6%
Laws: 9·11·12·14
// Real findings · Not fabricated

NUMBERS THAT
HOLD UP.

Every number below comes from real StatsBomb open data — 192 matches, 4,878 shot events. Verified by 5-fold cross-validation with honest reporting of model limits.

−9.7pp

Shootout vs in-game penalty gap — 65.0% (n=123) vs 74.7% (n=79). Real, measurable, persists across all 3 tournaments. 95% CIs overlap slightly at this sample size.

71.4%

Sudden-death kick conversion from 14 real kicks — counter-intuitively higher than regular shootout kicks (65.0%). Small sample, but real and unreported elsewhere.

110.87 → 103

StatsBomb's xG model expected 110.87 late-game goals — 103 were actually scored. Pressure suppresses finishing by ~7 goals beyond what shot quality alone predicts.

AUC 0.518

Pressure context predicts individual penalty outcomes barely above chance. Honest finding, not a failure — pressure shifts populations, not individuals. Consistent with sports science.

// Architecture

FIVE STEPS FROM
DATA TO INSIGHT

From StatsBomb's open JSON to IBM Granite narration — a clean, verifiable pipeline where every number traces back to a real source.

finemargins / pipeline
01

StatsBomb Open Data Fetch

192 matches pulled concurrently from StatsBomb's public GitHub repo — 2018 WC, 2022 WC, Women's WC 2023. No scraping, no paid API. Licensed under StatsBomb Public Data User Agreement.

StatsBomb JSON urllib.request ThreadPoolExecutor
02

Pressure Feature Engineering

202 penalty kicks and 1,228 late-game shots extracted. Pressure index built: shootout flag, sudden-death flag, knockout stage, stage weight, minutes remaining, xG residual.

pandas numpy parquet
03

Model Training + SHAP

Model A: Logistic Regression on 202 penalties (AUC 0.518 — honestly reported). Model B: Logistic Regression + xG baseline on 1,228 shots (AUC 0.773). SHAP LinearExplainer for per-kick attribution. 5-fold CV throughout.

scikit-learn shap 5-fold CV
04

IBM Tool Layer

IBM Docling parses the IFAB Laws of the Game 2025/26 PDF into structured chunks. IBM Context Forge MCP wires three typed tools (get_pressure_profile, get_law_text, get_overturn_rate) so Granite calls real data rather than generating statistics.

IBM Docling 2.103 IBM Context Forge MCP 1.0.3 FastMCP
05

IBM Granite Narration

IBM Granite (ibm/granite-4-h-small via watsonx.ai) narrates findings in three voice modes: fan, analyst, referee trainee. System prompts restrict Granite to numbers from tool calls only — no hallucination possible.

IBM Granite (watsonx.ai) ibm-watsonx-ai SDK Streamlit
